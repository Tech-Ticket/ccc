# 🏆 TechTicket Code Challenge Championship (CCC)

A professional training program designed to prepare students (ages 11–17) to **compete and win** in HTML & CSS web design competitions.

---

## 🚀 Overview

The **CCC (Code Challenge Championship)** is not a traditional course.

It is a **competition-focused training system** that transforms students from beginners into **confident competitors** by focusing on:

* ⚡ Speed & execution
* 🎯 Problem-solving under pressure
* 🧠 Requirement analysis
* 🎨 UI building using only HTML & CSS
* 🏆 Real competition simulations

---

## 🎯 What Students Learn

* Build real UI components (cards, layouts, forms)
* Master Flexbox & responsive design
* Create designs without images (pure CSS)
* Handle strict competition constraints
* Work under time pressure
* Think like a competitor

---

## 🧩 Program Structure

### 🟢 Level 1 — Builder (Sessions 1–5)

Learn core skills:

* CSS shapes & creative design
* Card systems & layout
* Product UI design
* Constraints handling
* Responsive design

---

### 🔵 Level 2 — Challenger (Sessions 6–9)

Train for competition:

* Speed & time management
* Real competition simulations
* Trap-based challenges
* Final championship exam
