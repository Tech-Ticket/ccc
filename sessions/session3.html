Session 3 — Advanced Product Card UI

🎯 Session Overview

Welcome back to Championship Training! In the last session, you mastered standard Flexbox cards. Today, we level up. We are building Premium Product Cards.

A standard card just holds text. A premium product card is an experience. It has layers, depth, vibrant gradients, and interactive hover effects. It looks like a high-end e-commerce store (like Nike or Apple).

In this session, you will learn to build:

  • The classic Two-Column Card Layout (Visuals on the left, Data on the right).
  • Linear Gradients to create stunning, modern backgrounds.
  • Layered Depth using box-shadow and border-radius.
  • Interactive Elements like color-selection circles and animated buttons.

🏆 The Competition Context

Why do we practice this? Because "Design a Product Page" is one of the most common tasks in web design competitions.

The Constraints (The Hard Part):

  • ❌ No Images: You cannot use an actual picture of a sneaker or a watch. You must simulate the "product image area" using CSS shapes, gradients, or typography.
  • ❌ No JavaScript: The hover effects and interactivity must be purely CSS.
  • ✅ Exact Match: The judges will check your spacing, contrast, and layout balance strictly.

🏗️ The Blueprint: Two-Column Layout

A premium product card isn't just a single box. It's usually a wide rectangle split perfectly into two halves using Flexbox.

The Two-Column Split

Product Visual Area
(Gradient Background)
HTML & CSS 
<!-- HTML Structure -->
<div class="product-card">
  <div class="card-visual">
    <!-- Gradient and shapes go here -->
  </div>
  <div class="card-info">
    <!-- Text and buttons go here -->
  </div>
</div>

/* CSS Layout */
.product-card {
  display: flex;
  width: 800px;       /* Make it wide */
  background: #ffffff;  /* Light background for contrast */
  border-radius: 20px;
  overflow: hidden;     /* Keeps children inside the rounded corners! */
}

.card-visual, .card-info {
  flex: 1; /* Forces both sides to be exactly 50% width */
}

If the parent .product-card has rounded corners (border-radius: 20px), but the child .card-visual has a solid background color, the sharp corners of the child will poke out of the parent's rounded corners!

overflow: hidden; acts like a cookie cutter. Anything that goes outside the parent's borders gets chopped off, keeping your corners perfectly smooth.

🎨 Advanced Styling: Gradients & Depth

Flat colors look outdated. Premium UI uses Linear Gradients for backgrounds and Box Shadows to lift the card off the page.

1. The Gradient Tool

A linear gradient transitions smoothly between two or more colors along a straight line. You define the angle (like 135deg) and the colors.

Change Colors:
CSS 
.card-visual {
  background: linear-gradient(135deg, #ec4899, #f97316);
}

2. Box Shadow Depth

To make the card pop off the dark background, we add a shadow. A modern shadow is soft, large, and slightly transparent.

.product-card {
  /* X-offset | Y-offset | Blur | Spread | Color */
  box-shadow: 0 20px 40px rgba(0, 0, 0, 0.4);
}

✨ Interaction: Hover Effects

A premium card needs premium buttons. When a user hovers over the "Add to Cart" button, it should respond smoothly.

Hover over me!

CSS 
.add-btn {
  background: #ffffff;
  color: #0f172a;
  border-radius: 50px; /* Pill shape */
  transition: all 0.3s ease; /* Crucial for smoothness */
}

.add-btn:hover {
  transform: translateY(-5px); /* Moves up */
  background: linear-gradient(135deg, #ec4899, #f97316);
  color: white;
  box-shadow: 0 10px 20px rgba(236, 72, 153, 0.4); /* Colored glow! */
}

🏋️‍♂️ Training Drills

Open your code editor. Build these components step-by-step.

Drill 1 (Easy): The Empty Frame

Create a <div class="product-card">. Give it a white background, width of 600px, height of 300px, and rounded corners (20px). Add display: flex; and overflow: hidden;.

.product-card { background: white; width: 600px; height: 300px; border-radius: 20px; display: flex; overflow: hidden; }

Drill 2 (Medium): The Split

Inside the card, add two divs: .visual and .info. In CSS, use flex: 1; on both so they share the space 50/50. Give the .visual div a linear gradient background from cyan to blue.

.visual, .info { flex: 1; }
.visual { background: linear-gradient(135deg, cyan, blue); }

Drill 3 (Hard): Color Options

Inside your .info div, create a color selection area. Create a flexbox container with 3 small circle divs inside it. Make one red, one blue, one black. Give them a hover effect that slightly enlarges them (scale).

Make the circles using: width: 25px; height: 25px; border-radius: 50%;
Hover: .circle:hover { transform: scale(1.2); cursor: pointer; }

🔥 Competition Task: Premium Sneaker Card

Brief: "Design a premium product card for a new sneaker release using HTML/CSS only."

Requirements:

  1. Use the Two-Column Layout. The overall card must have a light/white background.
  2. Left Side (Visual): A vibrant gradient background. Place large text inside saying "AIR MAX" rotated sideways to fake a graphic logo.
  3. Right Side (Info): Must include:
    • Product Category (e.g., "Men's Running Shoe" in small gray text).
    • Product Name (Large, bold, dark text).
    • Price (e.g., "$199" in a bright accent color).
    • Color selection circles (at least 3).
    • An "Add to Cart" button with a smooth hover effect.
  4. Add a large, soft box-shadow to the main card.
<!-- HTML Structure -->
<div class="product-card">
  
  <div class="visual">
    <h1 class="fake-logo">AIR MAX</h1>
  </div>
  
  <div class="info">
    <span class="category">Men's Running Shoe</span>
    <h2>Nike Air Max 270</h2>
    <h3 class="price">$199</h3>
    
    <div class="colors">
      <div class="circle red"></div>
      <div class="circle black"></div>
      <div class="circle blue"></div>
    </div>
    
    <button class="add-btn">Add to Cart</button>
  </div>
  
</div>

🏠 Homework: Level Up the Card

Take your Premium Sneaker Card and make it even better:

  • Typography: Use Google Fonts. Make the Product Name very bold (900), and the category text light and small.
  • Padding: Ensure the .info section has plenty of padding (like 40px) so the text doesn't touch the edges.
  • Responsiveness: Add a Media Query so that on mobile (max-width: 768px), the card changes from flex-direction: row to flex-direction: column, stacking the visual on top of the info!

📝 Session Summary

  • Premium Layouts: The split two-column design is a staple of modern product cards. Achieved easily by giving two child divs flex: 1 inside a flex container.
  • overflow: hidden; is required on the parent card to prevent child backgrounds from breaking out of rounded corners.
  • Gradients: linear-gradient(angle, color1, color2) adds massive visual appeal compared to flat colors.
  • Box Shadows: Soft, transparent shadows (rgba(0,0,0,0.5)) create the illusion of depth.
  • Interactivity: Hover effects using transform and transition make buttons feel alive.
