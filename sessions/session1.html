Session 1 — CSS Creative Shapes & Icon Design

🎯 Session Overview

Welcome to Championship Training! In normal web development, if you need an icon or a graphic, you just download an image file (PNG, SVG) and put it on the page.

In Coding Competitions, that is cheating.

Today, you will learn how to "fake" graphics. You will build complex visual designs, icons, and illustrations using only HTML boxes and CSS code.

You will learn:

  • How to manipulate the Box Model to draw Circles and Triangles.
  • How to use Gradients to make flat shapes look 3D.
  • How to use Positioning (Absolute/Relative) to stack shapes on top of each other.
  • How to use Box-Shadows to create depth.

🏆 The Competition Rules

If you enter a CSS Art or Web Design challenge, you must follow the strict rules of the platform:

  • ❌ NO Images allowed (No <img src="...">).
  • ❌ NO JavaScript allowed for drawing.
  • ✅ EVERYTHING must be drawn using HTML <div> tags and styled with CSS.

To win, you must think like an artist, but paint with code. Let's start with the basic geometric shapes.

📐 Drawing Basic Shapes

Every shape begins as a standard HTML <div>, which is a perfect square or rectangle. We use CSS to carve that block into different shapes.

1. The Perfect Circle

To turn a square into a circle, we use the border-radius property. If we set it to 50%, the corners curve so much they meet in the middle!

HTML & CSS 
<!-- HTML -->
<div class="circle"></div>

/* CSS */
.circle {
  width: 100px;
  height: 100px;
  background-color: #ec4899;
  border-radius: 50%; /* The magic line! */
}

2. The CSS Triangle Trick

How do we draw a triangle if a div only has 4 straight sides? We use a famous CSS hack involving thick, transparent borders.

CSS 
.triangle {
  /* 1. Shrink the actual box to nothing */
  width: 0;
  height: 0;
  background: transparent;
  
  /* 2. Create thick invisible borders on left and right */
  border-left: 50px solid transparent;
  border-right: 50px solid transparent;
  
  /* 3. Create a thick colored border on the bottom! */
  border-bottom: 100px solid #10b981;
}

When two very thick borders meet at the corner of a box, the browser cuts them at a 45-degree angle. By making the width and height of the box 0, the borders crash into each other. If we make 3 borders invisible (transparent) and 1 border colored, that angled cut creates a perfect triangle!

🎨 Gradients: Faking 3D Graphics

Flat colors are boring. To make an icon look like a real app icon, we use Gradients. A gradient smoothly blends one color into another.

/* Linear Gradient (Straight line) */
.app-icon {
  background: linear-gradient(135deg, cyan, blue);
}

/* Radial Gradient (Starts from center like a glowing orb) */
.planet {
  background: radial-gradient(circle at 30% 30%, pink, darkpurple);
}

Pro Tip: The circle at 30% 30% in the radial gradient creates a "fake lighting" effect by moving the bright center to the top-left corner!

🥞 Layering: Position Relative & Absolute

HTML elements normally stack on top of each other down the page. To draw an icon, we need shapes to sit inside and on top of each other like a stack of pancakes.

We do this using Positioning.

HTML & CSS 
<!-- HTML: The child goes INSIDE the parent -->
<div class="base-layer">
  <div class="top-layer"></div>
</div>

/* CSS */
.base-layer {
  position: relative; /* 1. Make parent the anchor point */
  width: 150px; height: 150px;
}

.top-layer {
  position: absolute; /* 2. Disconnect child from normal flow */
  top: 50%;           /* 3. Push down 50% from parent top */
  left: 50%;          /* 4. Push right 50% from parent left */
  transform: translate(-50%, -50%); /* 5. Perfect dead center! */
}

If you just use top: 50%; left: 50%;, the top-left corner of the child box will be in the middle, making it look off-center.

Adding transform: translate(-50%, -50%); pulls the child box back by exactly half of its own width and height, snapping it into perfect, mathematical center!

🏋️‍♂️ Training Drills

Open your code editor. Build these shapes. No copy-pasting allowed!

Drill 1 (Easy): The Target

Create a red square (100x100). Inside it, place a white circle (50x50). Center the circle perfectly inside the square using Absolute Positioning.

Parent needs: position: relative;
Child needs: border-radius: 50%; position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%);

Drill 2 (Medium): The Glowing Badge

Create a circle. Give it a linear-gradient background. Add a box-shadow that matches the color of the gradient so it looks like it's glowing in the dark.

box-shadow: 0 10px 20px rgba(0, 255, 255, 0.5); (Change the RGBA colors to match your gradient!)

Drill 3 (Hard): The Envelope Icon

Build an email envelope icon using ONE rectangle div (the envelope body) and ONE triangle div (the flap) positioned absolutely at the top.

Look back at the CSS Triangle Trick! Make the triangle point downwards by coloring the border-top and making left/right transparent. Position it top: 0; inside the rectangle.

🔥 Competition Task: Space App Icon

Brief: "Design a mobile app icon for a new Astronomy app using ONLY HTML and CSS."

Requirements:

  1. The main icon body must be a rounded square (border-radius: 20px).
  2. The background must be a dark space-like gradient (purple to black).
  3. Inside, draw a Planet (a circle with a radial gradient to make it look 3D).
  4. Bonus points: Draw a moon orbiting the planet (a small white circle positioned absolutely near the edge).

Remember, there are many ways to solve a CSS art challenge!

<!-- HTML Structure -->
<div class="app-icon">
  <div class="planet"></div>
  <div class="moon"></div>
</div>

/* CSS Setup */
.app-icon {
  position: relative;
  width: 120px; height: 120px;
  border-radius: 25px;
  background: linear-gradient(135deg, #1e1b4b, #000);
  box-shadow: 0 10px 20px rgba(0,0,0,0.5);
}

.planet {
  position: absolute;
  top: 50%; left: 50%;
  transform: translate(-50%, -50%);
  width: 60px; height: 60px;
  border-radius: 50%;
  background: radial-gradient(circle at 30% 30%, #ec4899, #8b5cf6);
}

.moon {
  position: absolute;
  top: 20px; right: 20px;
  width: 15px; height: 15px;
  background: #fff;
  border-radius: 50%;
}

  • It must contain at least 3 layered HTML elements (Parent, Child, Grandchild).

📝 Session Summary

  • CSS Shapes: Use border-radius: 50% for circles, and thick transparent borders to draw triangles.
  • Gradients: linear-gradient creates flat transitions, while radial-gradient creates glowing/3D sphere effects.
  • Layering: Use position: relative; on the parent to anchor it. Use position: absolute; on the children to stack them on top like stickers.
  • Absolute Center Trick: top: 50%; left: 50%; transform: translate(-50%, -50%);
