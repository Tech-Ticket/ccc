🏆 The Final Challenge Brief

TASK: Build a Complete "Premium Cloud Hosting" Landing Section.

A. Layout & Structure:
1. A Header containing the title: "Deploy to the Cloud".
2. A main section containing exactly 3 Pricing Cards in a single horizontal row.
3. The 3 cards must be for: "Basic", "Pro", and "Enterprise".
4. Each card MUST contain:
    • A Unicode icon at the top (e.g., ☁️, 🚀, 💎).
    • The Plan Title.
    • A Price.
    • A short description.
    • A "Choose Plan" button at the bottom.

B. Design & Styling:
1. The background of the entire page must be very dark (e.g., #0f172a).
2. The "Pro" card (the middle card) must stand out. Give it a gradient border or a glowing box-shadow.
3. Cards must have rounded corners (border-radius) and internal padding.

C. Interactivity & Responsiveness:
1. Hover Effect: When hovering over ANY card, it must lift up slightly and its button must change color. The transition must be smooth.
2. Mobile View: On screens smaller than 768px, the 3 cards MUST stack vertically.

⚠️ Remember: Judges are looking for equal card heights, clean alignment, and perfect button placement at the bottom of the cards!

🔍 The Champion's Solution

The timer is up. How did you do? Let's review the hidden difficulties in this challenge and how a master front-end engineer solves them.

Common Fatal Mistakes:

  • Unfinished Layout: Spending 45 minutes on the gradient for the "Pro" card and running out of time to make the cards responsive.
  • Button Misalignment: If the description text on the "Basic" card was shorter than the "Pro" card, the buttons wouldn't line up at the bottom. (Fix: display: flex; flex-direction: column; on the card, and margin-top: auto; on the button).
  • Ignoring Responsiveness: Not adding flex-wrap: wrap; and min-width, causing the cards to crush into tiny slivers on mobile phones.

The Master Code Architecture

index.html 
<header>
  <h1>Deploy to the Cloud</h1>
</header>

<section class="pricing-container">
  <!-- Basic Card -->
  <div class="card">
    <div class="icon">☁️</div>
    <h2>Basic</h2>
    <h3>$9/mo</h3>
    <p>Perfect for personal projects.</p>
    <button class="btn">Choose Plan</button>
  </div>
  
  <!-- Pro Card (Highlight) -->
  <div class="card pro-card">
    <div class="icon">🚀</div>
    <h2>Pro</h2>
    <h3>$29/mo</h3>
    <p>For growing businesses.</p>
    <button class="btn btn-pro">Choose Plan</button>
  </div>

  <!-- Enterprise Card (Similar to Basic) -->
</section>
style.css 
/* 1. The Responsive Flexbox Layout */
.pricing-container {
  display: flex;
  gap: 20px;
  flex-wrap: wrap; /* Handles mobile stacking */
  align-items: stretch; /* Equal height cards! */
}

/* 2. The Smart Card Structure */
.card {
  flex: 1;
  min-width: 280px; /* Mobile breakpoint threshold */
  background: #1e293b;
  padding: 30px;
  border-radius: 12px;
  
  /* Inner Flexbox to push button to bottom */
  display: flex;
  flex-direction: column;
  transition: transform 0.3s, box-shadow 0.3s;
}

.btn {
  margin-top: auto; /* The Magic Trick */
  transition: background 0.3s;
}

/* 3. The Pro Highlight & Hover Effects */
.pro-card {
  border: 2px solid #06b6d4;
  box-shadow: 0 0 20px rgba(6,182,212,0.2);
}

.card:hover {
  transform: translateY(-10px);
}
.card:hover .btn {
  background: #06b6d4; color: #000;
}
